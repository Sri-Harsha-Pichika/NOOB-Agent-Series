# Security Policy

## Supported Versions
We maintain the latest `main` branch. Security fixes will be applied to current releases.

## Reporting a Vulnerability
If you discover a security vulnerability in NOOB Agents:
1. **Do not open a public issue.**
2. Email the maintainers directly at:  
   **[sriharsha791@gmail.com]**
3. Provide:
   - Description of the vulnerability.
   - Steps to reproduce.
   - Potential impact.
   - Suggested mitigation if available.

We will:
- Acknowledge receipt within 48 hours.
- Investigate and confirm the issue.
- Provide updates on remediation progress.
- Credit you in release notes if desired.

## Responsible Disclosure
We ask that you:
- Do not exploit vulnerabilities.
- Do not share details publicly until a fix is released.
- Work with us to protect the community.

---

Thank you for helping keep NOOB Agents secure!
